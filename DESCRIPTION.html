Probably the sharpest git repository organizer and rebase/merge workflow automation tool you've ever seen! This plugin adds an extra tab in the Git tool window that gives the instant answer to the questions: what are the branches in this repository? what needs be merged/rebased/pushed/pulled?

The bird's eye view provided by Git Machete makes rebases/push/pulls hassle-free even when multiple branches are present in the repository: master/develop, your topic branches, teammate's branches checked out for review etc.

Along these lines, Git Machete helps in creating and maintaining small, focused, easy-to-review PRs.

You can read more at the plugin's GitHub repository, as well as at the git-machete CLI GitHub repository.
